## Important SEO Implementation Instructions

Do not add every keyword to the homepage. Each major service should have
its own dedicated page targeting one primary keyword, several closely
related secondary keywords, and relevant long-tail search queries.

Avoid keyword stuffing. Use keywords naturally in page titles, meta
descriptions, H1 and H2 headings, introductory paragraphs, service
descriptions, image alt text where genuinely descriptive, internal
links, FAQ content, case studies, and blog content.

For stronger Google visibility, implement technical SEO, XML sitemap,
robots.txt, canonical tags, structured data/schema markup, Google Search
Console, Google Analytics, fast Core Web Vitals, responsive design,
optimized images, strong internal linking, original service content,
portfolio pages, case studies, reviews, authoritative backlinks, and
location-specific landing pages only for locations genuinely served.

SEO does not guarantee a top Google position. Ranking depends on search
competition, content quality, domain authority, backlinks, user
experience, technical performance, search intent alignment, and
consistent ongoing optimization.
